# SMPA Course Materials Archive

This repository is a consolidated snapshot archive of public course-material repositories from GW SMPA-related R/data journalism classes and workshops.

Each original repository was copied into its own subdirectory. This archive does not preserve the original Git histories; the original archived repositories remain the source of record for commit history, forks, stars, and existing links.

## Directory Layout

```text
2021-spring/
  sessions/

2022-fall/
  assignments/
  sessions/

2023-spring/
  assignments/
  sessions/

2023-fall/
  assignments/
  sessions/

2025-fall/
  sessions/
```
